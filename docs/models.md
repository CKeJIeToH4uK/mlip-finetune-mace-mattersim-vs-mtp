# Модели

В документации сохраняются технические имена: MTP, MACE, MatterSim, MAE и RMSE. Для MoNbTaVW базовый MACE и протокол MACE-EW50 с увеличенным весом ошибки энергии фиксируются как разные строки результатов.

Файлы весов моделей и checkpoints физически не включены в репозиторий; ссылки на их исходные расположения зафиксированы в манифестах.
