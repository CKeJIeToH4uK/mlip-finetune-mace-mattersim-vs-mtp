# Метрики и единицы

- ошибки энергии измеряются в `meV/atom`;
- ошибки сил измеряются в `meV/Å`;
- основные метрики: `MAE` и `RMSE`.

Репозиторий не меняет единицы измерения, обозначения разбиений, алгоритмы агрегации или финальные численные значения. Основной внешний файл для написания отчета — `../text_coursework/RESULTS_SOURCE_OF_TRUTH.md`; локальные копии проверенных таблиц лежат в `results/verified_metrics/`.
