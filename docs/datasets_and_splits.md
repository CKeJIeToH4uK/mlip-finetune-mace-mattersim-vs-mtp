# Датасеты и разбиения данных

## organic-датасет

Основные результаты соответствуют одной оценке переноса между температурами: 300K, 600K и 1200K. MTP ensemble используется только как sanity check для train/valid разбиений при 300K.

## H2O

Основные результаты относятся к отложенной validation-выборке. Train-строки могут использоваться для проверки происхождения результатов и согласованности таблиц, но не смешиваются с основными validation-результатами.

## MoNbTaVW

Основные результаты также относятся к отложенной validation-выборке. Базовый MACE и протокол MACE-EW50 учитываются как разные варианты настройки. Строка MTP-16 используется с отдельной оговоркой.
