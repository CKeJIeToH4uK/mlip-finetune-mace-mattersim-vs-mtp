# Ограничения

- Из этого репозитория нельзя полностью переобучить модели с исходных данных.
- Исходные датасеты, веса моделей и чекпоинты не включены физически; их роль и расположение описаны в манифестах.
- Старые контексты сохранены только как архивные материалы и не могут переопределять `verified` CSV.
- MTP ensemble используется только как дополнительная проверка для organic 300K и не является основной строкой результатов.
- FLiNaK находится вне основной линии экспериментов.
