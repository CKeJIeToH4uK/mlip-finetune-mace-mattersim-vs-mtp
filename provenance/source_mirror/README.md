# Историческое зеркало исходных скриптов

Эта папка содержит исторические копии скриптов и запусков, по которым можно проверить происхождение результатов и историю экспериментов.

Исполняемая часть репозитория находится в `src/mlip_benchmark/`. Скрипты в этой папке могут содержать старую структуру запусков, пути к кластерному окружению или команды, требующие внешних исходных данных и чекпоинтов, описанных в манифестах.

Используйте эту папку для аудита и восстановления контекста экспериментов, а не как чистую точку запуска.
