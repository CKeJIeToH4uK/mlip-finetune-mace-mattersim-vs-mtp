# Historical Source Mirror

This directory is a provenance-only mirror of useful coursework scripts and run files. It is not the runnable package surface of `coursework_clean/`.

Runnable/review helper modules live under `src/mlip_benchmark/`. Historical scripts here may contain cluster-specific paths, old job layouts, or commands that require external raw data/checkpoints documented in manifests.

Use this mirror to inspect provenance and experiment history, not as a clean execution entrypoint.
