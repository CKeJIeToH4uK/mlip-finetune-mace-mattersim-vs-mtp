# Историческое зеркало исходных скриптов

Эта папка хранит копию полезных скриптов и файлов запусков из рабочей директории курсового проекта. Она нужна только для проверки происхождения результатов и истории экспериментов.

Исполняемая часть репозитория находится в `src/mlip_benchmark/`. Скрипты в этой папке могут содержать старую структуру запусков, пути к кластерному окружению или команды, требующие внешних исходных данных и чекпоинтов, описанных в manifests.

Используйте эту папку для аудита и восстановления контекста экспериментов, а не как чистую точку запуска.
