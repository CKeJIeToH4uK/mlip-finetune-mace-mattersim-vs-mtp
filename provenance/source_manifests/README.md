# Манифесты источников

CSV-файлы в этой папке фиксируют происхождение материалов для очищенного репозитория.

- `included_manifest.csv`: скопированные и сгенерированные артефакты очищенного репозитория, кроме управляющих файлов, которые создавали бы циклические записи в манифесте.
- `excluded_heavy_manifest.csv`: исходные датасеты, чекпоинты, веса моделей, результаты сборки, бинарные файлы и другие тяжелые артефакты, намеренно не включенные физически.
- `archive_external_manifest.csv`: старые контексты, внешние зоны, учебные папки, архивы и справочные материалы, которые не копировались целиком.

## Исключения для самоссылок

Манифесты могут отсутствовать в `included_manifest.csv`; это сделано, чтобы избежать самоссылочных строк. `checksums.sha256` намеренно исключает сам себя и покрывает остальные манифесты вместе с обычными скопированными или сгенерированными артефактами.

## Каноническая схема

Текущие CSV-заголовки являются каноническими для очищенного репозитория. Ранние поля из плана сборки, такие как `kind`, `archive_status` или `replacement_reference`, представлены текущими колонками `category`, `artifact_type`, `possible_relevance`, `reason_not_copied` и `notes`, где это применимо.
