# Scripts

- `validate_manifests.py`: validates manifest coverage, excluded artifact policy and checksums.
- `validate_results.py`: validates required result tables and split terminology guardrails.
- `prepare_metrics.py`: lists curated CSV tables and row counts.
- `make_tables.py`: creates small Markdown previews for verified CSV tables.
- `make_figures.py`: lists selected report figure assets.

Historical source training/evaluation scripts are mirrored under `provenance/source_mirror/` and are provenance-only.
