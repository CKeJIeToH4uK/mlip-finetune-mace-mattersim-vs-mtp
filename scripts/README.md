# Scripts

- `validate_manifests.py`: проверяет покрытие manifest-файлами, политику исключенных артефактов и контрольные суммы.
- `validate_results.py`: проверяет обязательные таблицы результатов и ограничения на терминологию разбиений.
- `prepare_metrics.py`: выводит список отобранных CSV-таблиц и количество строк.
- `make_tables.py`: создает небольшие Markdown-превью для проверенных CSV-таблиц.
- `make_figures.py`: выводит список отобранных графиков для отчета.

Исторические training/evaluation-скрипты зеркалированы в `provenance/source_mirror/` и используются только для проверки происхождения результатов.
