# mlip_benchmark

Эта папка содержит легкие вспомогательные модули для просмотра и проверки очищенного репозитория.

Исторические скрипты курсового проекта вынесены в `provenance/source_mirror/` и используются только для сверки происхождения результатов. Они не входят в основной исполняемый вспомогательный пакет.

Модули в этой папке не переобучают модели. Они индексируют таблицы, графики и правила работы с разбиениями данных для проверки репозитория и подготовки отчета.
