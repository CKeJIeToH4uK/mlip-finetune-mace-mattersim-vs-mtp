# Curated Coursework Repository

Этот репозиторий собирает проверяемую часть материалов курсового проекта: финальные метрики, таблицы для отчета, графики, скрипты проверки, provenance-артефакты и архивный контекст.

Крупные исходные данные, веса моделей, чекпоинты, тяжелые архивы, локальные бинарные файлы, вложенные репозитории и результаты сборки не включены физически. Их расположение и роль зафиксированы в `provenance/source_manifests/`.

Финальные численные результаты для отчета зафиксированы во внешнем рабочем документе `text_coursework/RESULTS_SOURCE_OF_TRUTH.md`. В этом репозитории ему соответствуют проверенные CSV-файлы по отдельным системам в `results/verified_metrics/`: organic-датасет, H2O и MoNbTaVW. Таблицы из `provenance/handoff_tables/` используются только для дополнительной сверки происхождения результатов и не являются полным источником финальных чисел.

Основные системы и модели: organic-датасет, H2O, MoNbTaVW; MTP, MACE, MatterSim и energy-weighted протокол MACE для MoNbTaVW.

Для навигации по папкам см. `PROJECT_STRUCTURE.md`; границы локальной проверки и список намеренно исключенных активов описаны в `REPRODUCIBILITY.md`.

## Быстрая карта

- `docs/` — краткие методологические описания: постановка задачи, данные, протокол экспериментов, модели, метрики, результаты, происхождение результатов и ограничения.
- `results/` — проверенные таблицы метрик, вспомогательные таблицы, дополнительные проверки и отобранные графики.
- `provenance/` — материалы происхождения результатов: handoff tables, source mirror, log snippets, slurm scripts, manifests, checksums и raw eval refs.
- `src/mlip_benchmark/` — легкие вспомогательные модули для анализа, проверки результатов и построения таблиц/графиков.
- `scripts/` — скрипты проверки и вспомогательные команды для работы с репозиторием.
- `presentation_archive/` — финальная презентация и выбранные preview/backup материалы.
- `archive/` — старые контексты и исключенные ветки, не являющиеся источником финальных чисел.

## Проверка

Запуск:

```bash
python scripts/validate_manifests.py
python scripts/validate_results.py
```

Эти скрипты проверяют покрытие manifest-файлами, список исключенных артефактов, наличие обязательных файлов с метриками, терминологические ограничения и контрольные суммы скопированных материалов.
