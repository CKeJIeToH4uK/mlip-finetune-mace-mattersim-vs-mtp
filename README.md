# Проверяемый репозиторий курсового проекта

Этот репозиторий собирает проверяемую часть материалов курсового проекта: финальные метрики, таблицы для отчета, графики, скрипты проверки, материалы происхождения результатов и архивный контекст.

Крупные исходные данные, веса моделей, чекпоинты, тяжелые архивы, локальные бинарные файлы, вложенные репозитории и результаты сборки не включены физически. Их расположение и роль зафиксированы в `provenance/source_manifests/`.

Финальные численные результаты для отчета зафиксированы во внешнем рабочем документе `../text_coursework/RESULTS_SOURCE_OF_TRUTH.md`, который намеренно не входит в очищенный репозиторий. В этом репозитории ему соответствуют проверенные CSV-файлы по отдельным системам в `results/verified_metrics/`: organic-датасет, H2O и MoNbTaVW. Таблицы из `provenance/handoff_tables/` используются только для дополнительной сверки происхождения результатов и не являются полным источником финальных чисел.

Основные системы и модели: organic-датасет, H2O, MoNbTaVW; MTP, MACE, MatterSim и протокол MACE с увеличенным весом ошибки энергии для MoNbTaVW.

Для навигации по папкам см. `PROJECT_STRUCTURE.md`; границы локальной проверки и список намеренно исключенных активов описаны в `REPRODUCIBILITY.md`.

## Быстрая карта

- `docs/` — краткие методологические описания: постановка задачи, данные, протокол экспериментов, модели, метрики, результаты, происхождение результатов и ограничения.
- `results/` — проверенные таблицы метрик, вспомогательные таблицы, дополнительные проверки и отобранные графики.
- `provenance/` — материалы происхождения результатов: сопроводительные таблицы, историческое зеркало исходников, фрагменты логов, slurm-скрипты, манифесты, контрольные суммы и небольшие файлы исходной оценки.
- `src/mlip_benchmark/` — легкие вспомогательные модули для анализа, проверки результатов и построения таблиц/графиков.
- `scripts/` — скрипты проверки и вспомогательные команды для работы с репозиторием.
- `presentation_archive/` — финальная презентация, материалы предпросмотра и резервные материалы.
- `archive/` — старые контексты и исключенные ветки, не являющиеся источником финальных чисел.

## Проверка

Запуск:

```bash
python scripts/validate_manifests.py
python scripts/validate_results.py
```

Эти скрипты проверяют покрытие манифестами, список исключенных артефактов, наличие обязательных файлов с метриками, терминологические ограничения и контрольные суммы скопированных материалов.
